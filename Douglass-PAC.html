Frequently-Asked Questions (FAQs)

The following FAQs are intended to provide a bare-bones guide to getting started with Google tools. There are plenty of in-depth tutorials for all of these tools on the web, especially in Youtube. This document is customized to the needs of our group and should help the inexperienced user get started.

Important: If anyone needs help with any of this contact me at: john.tibbetts@kinexis.com.

General

The Douglass Neighborhood Political Action Group (Douglass-PAG) will use Google Groups, Docs, and Sheets to enable us to work together. We use them for group email communication, document creation, storage, spreadsheets etc. These collaboration tools are widely used, free, and can be run using any standard email client and web browser.

Do I have to sign up with Google to use them?

The simplest use of these tools do not require any sign-up. Simple use includes:

  • Sending an email to the whole group

  • Viewing group documents or spreadsheets that have been marked public

However full tool use does require that you use or obtain a Google account. Full use includes:

  • Creating or modifying documents or spreadsheets.

  • Commenting on documents

  • Reviewing historical email threads

  • And much more

Where do I get a Google account?

First, If your email address ends with ‘@gmail.com’ you already have a Google account. Simply use your gmail address when logging in. Use the password you usually use.

If you don’t have a gmail account, it’s easy to get one at the Google signup page and follow the instructions. By default this page will create a new gmail account. It doesn’t cost anything. You don’t need to use this for email, but it will however provide you with access to Google tools.

If you prefer using your current email address and don’t want a gmail account, press the link "I prefer to use my current email address". Then enter that email address.

Important: If you use this approach choose an entirely new password for this account--not the one currently use for email. It’s always a security risk to use the same password for multiple sites.

Either approach--signing up for gmail or using your current email--creates a Google account that we’ll use for connecting to our tools.

Using Google Groups

Google Groups allows us to send an email to our whole group, review emails that have already been sent, and provide numerous other services for community conversations.

How to I send an email to our Douglass-PAG group?

  1. Create a standard email

  2. Address it to ‘douglass-pag@googlegroups.com’

  3. Send

The email is immediately sent to all members of the group. It is also recorded in a forum that any member can review.

How can I see an archive of all emails that have been sent in our Douglass-PAG group?

  1. In a web browser go to https://googlegroups.com

  2. (Sign in with your google credentials if the "Sign In" button appears on the top right)

  3. Click on the "My groups" button, then select “Douglass Neighborhood Political Action Group”

  4. From here you’re in a forum where you can view messages, filter them, organize by topic, etc.

Alternatively: you can bookmark the following URL to jump right into the forum:

https://groups.google.com/forum/#!forum/douglass-pag

How to unsubscribe from the group

  1. In a browser go to the Douglass-PAG forum as described in the previous FAQ

  2. Find the "My Settings" pull-down on the top right of the page:

  3. Click the down-arrow and select "Leave this group"

How can an interested neighbor join the group?

  1. Tell your neighbor to open a web browser on their own device

  2. Put in the URL: https://groups.google.com/forum/#!forum/douglass-pag/join

  3. A small form appears

    1. Verifying your email address

    2. Prompts for some simple options

    3. And allows him/her to send a message to the list manager (introduce yourself and verify that he/she lives in the neighborhood)

  4. Hit the button "Apply to join this group"

  5. An email will be sent to the group managers and someone will approve the membership

Using Google tools

Google offers a number of applications free-of-charge to Google users. At the moment we will focus on three central tools:

  1. Google Drive Drive is a storage area for storing and sharing any kind of computer file. It’s much like Dropbox.

  2. Google Docs Docs is a web-based word processor. It’s considerably easier and less function-rich than Microsoft Word but satisfies the same need.

  3. Google Sheets Sheets is a web-based spreadsheet. Like Docs, it’s easier and less function-rich than the Microsoft-equivalent Excel.

Google Docs and Sheets provide considerably more capability for allowing multiple people to work in a document at the same time. It also has a rich mechanism for adding comments to a document (or spreadsheet) and for automatically storing all previous versions of it.

How do I get access to the Douglass-PAG ‘shared folder’?

Google drive works like any file system: it contains files (of any kind) and folders that contain files or other folders. Douglass-PAG has a pre-created folder that is a ‘shared folder’. This means that the folder is pre-set to be used by everyone enrolled in the Douglass-PAG Google Group. In other words, once you’re in the group you have access to the folder.

To access the Douglass-PAG shared-folder, click on the link:

https://drive.google.com/open?id=0B3ktxD2sFKPWLVhkS2FTSUcxSE0

It might be a good idea to cut-and-paste this URL into a browser bookmark for easier subsequent access.

What will go into the Douglass-PAG folder?

Anything we want to put there. We anticipate things like:

  • Documents about the resistance; for example, the Indivisible Guide is already there.

  • Individual statements of any kind

  • Spreadsheets of resistance resources like collections of URLs for important websites

  • Spreadsheets of contact information for state and federal legislators, senate committees.

  • Sample text of letters or messages for a campaign

Does everyone get to add or modify documents? Won’t that cause chaos?

Google tools were created with collaboration in mind. Several features help out with this:

  • Changes that anyone makes are instantly backed up into a ‘revision history’. Even if a large chunk of a document is deleted it can be later viewed and recovered.

  • Documents can be created with many kinds of protection. A document creator (owner) might want to ensure that no one else can make changes, but everyone can read it. Alternatively a sub-group can collectively work on the document, even at the same time.

This isn’t to say that we shouldn’t be careful with documents, especially documents owned by someone else. Though recovery is always possible it can still be tedious. Our general approach will be to start out with pretty open access and then see how it all goes.

How do I create a Google Doc (text)?

Within the Douglass-PAG folder you will see a blue "New" button at the top left-hand side. Clicking on the button opens a drop-down. Select “Google Docs”, then “Blank document”. (or “from Template” if desired).

How do I create a Google Sheet (spreadsheet)?

Within the Douglass-PAG folder you will see a blue "New" button at the top left-hand side. Clicking on the button opens a drop-down. Select “Google Sheets”.

How do I learn these tools?

Anyone with experience with a word processor or spreadsheet should be able to get along without much assistance. Beyond that there are hundreds of online resources: tutorials in both print and video.

Will we use other tools than those mentioned above?

We will use any tool that advances our objectives. But those mentioned above will be our central tools.
